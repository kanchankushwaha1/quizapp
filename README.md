# Quiz Application

A **web-based quiz application** built using **Spring Boot** and **MySQL**.  
This application allows users to register, take quizzes, and view their results. Admins can create and manage quizzes and questions.

---

## Features

- User registration and login
- Admin panel to create and manage quizzes
- Multiple-choice quizzes
- View scores and quiz history
- Data stored in MySQL database

---

## Tech Stack

- **Backend:** Java, Spring Boot
- **Database:** MySQL
- **Frontend:** HTML, CSS, JavaScript (Thymeleaf or any template engine)
- **Build Tool:** Maven
